Understanding Specificity

Specificity

  • Spcificity is the means by which browsers decide which CSS property values are the most relevant to an element and, therefore, will be applied.
  • There are 4 types of specificity thatare added up
    • Inline = a
    • IDs = b
    • Classes = c
    • Types (elements) = d
  • Added by this situation:
    • a,b,c,d
    • A = 1000, B = 100, C = 10, D = 1

Types

  • Types are classified as:
    • Elements
      • i.e.: h2, p, etc
  • Code in CSS:
    • h2 { color:green; }
  • This will change ALL h2 elements
